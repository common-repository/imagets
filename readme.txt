= Collect images =
ImageTS can collect images on web pages, included Google image search, blogs, news websites, Instagram, Pinterest, Tumblr and Flickr.

= Convert images =
You can crop, resize, add filter or logo with ImageTS. Also you don't need to download images and upload them to your server particularly.

= More contents, more visitors =
Your website and blog is more powerful with ImageTS. ImageTS can handle all of your needs about images.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
